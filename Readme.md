# MCP Java App

This is a simple MCP (Model Context Protocol) server built with Java and Spring Boot. It exposes two methods as MCP tools using the `spring-ai-mcp` library.

## MCP Tools

### 1. `Get Product`
Return details about a product by name.

### 2. `Get All Products`
Return details about a all products.

### 2. `Search Knowledge`
Return details customers, products, invoices from **RAG**

## MCP Server UI
To test the tools interactively, you can use the MCP Web UI:
```
npx @modelcontextprotocol/ui@latest
```
